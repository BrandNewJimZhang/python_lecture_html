Chapter 3 基本语法与基本数据类型

3.1 基本语法

3.1.1 变量

变量名必须是大小写英文、数字和_的组合，且不能用数字开头。

>>> a = 1
>>> a1 = 2
>>> _a = 3
>>> A = 4

（常量）

3.2 基本数据类型及其基本操作

整数、复数、浮点数、布尔值、None、字符串、列表、元组、字典、集合

3.2.1 整数

整数是没有小数部分的数字，可以是正数、负数或零。

>>> a = 1

3.2.2 复数

复数由实数部分和虚数部分构成，可以用a + bj或complex(a,b)表示， 复数的实部a和虚部b都是浮点型。

>>> a = 1 + 2j
>>> b = complex(1, 2)

3.2.3 浮点数

浮点数由整数部分与小数部分构成，浮点数运算可能会有四舍五入的误差。

>>> a = 1.1

3.2.4 布尔值

布尔值只有True、False两种值，要么是True，要么是False，在Python中，可以直接用True、False表示布尔值（请注意大小写），也可以通过布尔运算计算出来。

>>> a = True
>>> b = False

3.2.5 None

None是Python里一个特殊的类型，用来表示什么都没有，它只有一个值，就是None。

>>> a = None

3.2.6 字符串

字符串是以单引号或双引号括起来的任意文本，比如’abc’，“xyz”等等。

>>> a = 'abc'
>>> b = "xyz"

3.2.7 列表

列表是一种有序的集合，可以随时添加和删除其中的元素。

>>> a = [1, 2, 3]

3.2.8 元组

元组是另一种有序的列表，但是元组一旦初始化就不能修改。


>>> a = (1, 2, 3)

3.2.9 字典

字典是一种无序的集合，使用键-值（key-value）存储，具有极快的查找速度。

>>> a = {'a': 1, 'b': 2, 'c': 3}

3.2.10 集合

集合是一种无序的不重复元素序列。

>>> a = {1, 2, 3}
